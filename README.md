# DeepRead Community

Welcome to the official community hub for [DeepRead](https://www.deepread.tech/).

## What is DeepRead?

DeepRead is a production-ready document processing API that converts scans, PDFs, and images into structured data with human-level accuracy. When uncertain, it flags documents for review instead of guessing.

**Key capabilities:**
- Multi-model consensus OCR (parallel processing for accuracy)
- AI Schema Optimizer that trains on your documents
- Per-field confidence scoring with automatic human review flags
- 95% accuracy on your documents, not just benchmarks

## What This Community Is For

This is a space for developers integrating DeepRead to:

- **Ask questions** about API usage, schemas, and pipelines
- **Share challenges** you're facing with document processing
- **Discuss solutions** and best practices with other developers
